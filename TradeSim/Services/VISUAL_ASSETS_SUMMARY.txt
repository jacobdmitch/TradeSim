🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Scripts won't run
  └─ Solution: ./make_executable.sh

Problem: Loading screen doesn't show
  └─ Solution: Clean build (Cmd+Shift+K), rebuild

Problem: Icon is blank
  └─ Solution: Generate icons first with ./generate_icons.sh

Problem: Animations choppy
  └─ Solution: Normal in Simulator, test on real device

Problem: Build failed
  └─ Solution: Open Xcode, check for errors, clean build
